SymPy webpages
==============

These are SymPy webpages served at:

http://sympy.org/

using github. Currently it only redirects to the sympy googlecode page, until
we create some nice pages.

Web in development
------------------

Webpages in development are here:

http://sympy.org/dev/

to modify them, do::

    cd dev
    # edit the pages
    ./generate
